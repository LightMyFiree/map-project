# Map Project (статическая карта)

Проект — статичный сайт (HTML/CSS/JS) с интерактивной картой Leaflet. Он подходит для GitHub Pages: **не нужен сервер**.

## Python в дипломе: генератор данных

Чтобы удобно наполнять карту и при этом выполнить требование «использовать Python», в проект добавлен небольшой **data pipeline**:

- **Источник**: `data_preparation/places.csv`
- **Результат**: `data/points.generated.geojson` (отдельный файл, чтобы не перезаписать рабочие данные)

### Запуск генерации

Из корня проекта:

```bash
python3 data_preparation/build_map_data.py
```

Скрипт:
- не требует `pip` и внешних библиотек;
- валидирует обязательные поля;
- пишет координаты в GeoJSON-формате \([lng, lat]\) — это важно.

Чтобы применить новые данные на сайте, замените файл данных:

```bash
cp data/points.generated.geojson data/points.geojson
```

## Структура

- `index.html` — разметка
- `style.css` — стили (включая темы)
- `script.js` — логика карты, фильтры, импорт/экспорт, маршрутизация
- `data/points.geojson` — данные карты
- `data_preparation/` — исходные данные (CSV) + Python-генератор
